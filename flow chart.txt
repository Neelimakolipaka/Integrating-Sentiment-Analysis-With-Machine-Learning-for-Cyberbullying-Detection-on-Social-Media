Dataset: https://www.kaggle.com/datasets/andrewmvd/cyberbullying-classification?select=cyberbullying_tweets.csv

Notebook:
=========

1. Importing the packages
2. Exploring the dataset
	- load the cyberbullying dataset
3. Pre-preprocessing
	- convert the all text lowercase
	- emoji to text
	- remove URLS and scpecial characters
	- Tokenization
	- Stopwords
	- Lemmatization
4. Data Visualization
	- Distribution of Cyberbullying Types
	- Wordcloud
5. TF-IDF Vectorizatoin for input text column
6. Convert the string to integer using labing encoding 
7. Resampling the data
	- Oversampling using SMOTE
8. Train-Test split
9. Build the Model with apply GridSearchCV
	- 
	- Logistic Regression
	- RandomForest
	- XGBoost
	- DecisionTree
	- Naive Bayes
	- SVM 
	- ExtraTree
	- GradientBoost
	- AdaBoost
	- Voting Classifier ( MLP + Bagging with LR)
10. Visualize the confusion matrix for each models	
11. Comparing the models with accuracy, precision, recall, f1 score
12. Comparison Graphs are generated with scores of all models.
13. Save the best trained model using joblib
14. XAI used to explain feature dependincy

Extension:
------------
This project presents a cyberbullying detection framework that integrates sentiment analysis with optimized machine learning algorithms to improve detection accuracy on social media. By applying effective text preprocessing and handling class imbalance, the Extra Tree classifier achieved the best performance with 95.38% accuracy.

As an extension, we incorporated SMOTEENN along with an additional ensemble technique to achieve better performance.

We also applied XAI methods such as LIME and SHAP to explain feature importance.

Furthermore, we implemented a Voting Classifier( MLP + Bagging with LR) achieved highest accuracy 97.4% to make the model more robust and improve prediction accuracy.
